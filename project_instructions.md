# Project Instructions - BullDogAlpha

This file contains key instructions and guidelines for the **BullDogAlpha** project.

## Third-Party Platform Integrations (One CLI)

The One CLI (`one`) is installed on this machine. It allows interaction with over 250+ third-party platforms through their APIs.

### Supported Platforms
- **Communication & Collaboration**: Slack, Gmail, Teams, Discord, Zoom
- **E-commerce & Payments**: Shopify, Stripe, PayPal, HubSpot
- **Developer Tools**: GitHub, GitLab, Jira, Confluence, Linear
- **And many more (250+ platforms)**

### Usage Guideline
Whenever you need to interact with any third-party platform or external service, use the `one` skill/command rather than implementing custom API clients from scratch.
